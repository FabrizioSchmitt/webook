Listas

Como as listas funcionam

As listas no html surgem em aspectos semânticos e visuais em uma única sintaxe.Define-se o tipo de lista, e dentro dessa lista são inseridos itens. Cada item dessa lista representa uma marcação seja do tipo definido pelo tipo da lista a ser usado.

OBS
A partir do html 5 a tag <li> passou a ser uma tag vazia. Ou seja, não existe mais a necessidade de utilizar uma tag de fechamento </li>.

Lista ordenada <ol>

Listas ordenadas inserem enumeradores com base no valor do item anterior, adicionando a lista (no fim) valores incrementais.
Exemplo:

  1. Melância
  2. Melância
  3. Melância
Observe o valor [1.,2.,3] no inicio de cada item da lista.

Lista não ordenada <ul>

Funciona exatamente da mesma forma que uma lista ordenada, exceto que cada item inserido na lista possui um marcador gráfico que não é enumerado, e não dispõe de semântica ordenativa.
Exemplo:

  • Melância
  • Melância
  • Melância

Lista de descrições

As listas descritivas oferecem uma maneira de criar descrições pré-formatadas para determinados assuntos.

A estruta de uma lista de descrições é muito semelhante a estrutura criada para uma tabela em HTML

Início da descrição do assunto
Título ou subtítulo do assunto.
Dados sobre assunto.

Listas aninhas

É possível aninhar listas do mesmo tipo, ou de tipos diferentes. Esse, é um dos principais princípios no design e implementação de padrões de menus em HTML.

Trabalhando com o atributo TYPE em listas

Além dos atributos normais que cada uma das listas podem ter, uma vez que essas tags herdam atributos da tag html, e outros atributos as quais possam eventualmente herdar características, existem alguns atribos próprios para trabalhar listas.

Idependente de qual a lista que estejamos trabalhando, o atributo type é o operador das moficações utilizadas em listas.

Atributos da listas ordenadas

Listas ordenadas possuem 5 valores para o atributo type, o numérico (como ja visto anteriormente) e outros 4, são eles:

Do tipo alfebético minúsculo;

  1. item 1
  2. item 2
  3. item 3
  4. item 4

Do tipo alfabético maiúsculo:

  1. item 1
  2. item 2
  3. item 3
  4. item 4

Do tipo algarismo romano menor

  1. item 1
  2. item 2
  3. item 3
  4. item 4

Do tipo algarismo romano maiúsculo

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Atributo Start

As listas ordenadas oferecem ainda um outro tipo de atributo, o atributo: start. O atributo start determina (dentro de uma lista ordenada) qual o valor inicial da contagem enumerada da lista.

  1. item 1
  2. item 2
  3. item 3

Atributos de listas não ordenadas

As listas não ordenadas trabalham com os 3 seguintes valores para type:
  • type="square" Marcador quadrado
  • type="disc" Disc, o marcador padrão.
  • type="circle" Marcador de circulo vazado
