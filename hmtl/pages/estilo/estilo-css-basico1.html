Os tipos de aplicação CSS

As css(as folhas de estilo) podem ser aplicadas de formas diferentes. A seguir veremos as formas mais comuns:

CSS inline

A aplicação inline (em linha) do CSS como o próprio nome já diz, é aplica linha por linha do documento, ou seja: para cada elemento presente que se queira mudar, a aplicação de estilo terá que ser feita manualmente com base no elemento style.

CSS interno

A segunda forma de uso do CSS é através da aplicação por documento. Essa forma de aplicação tem como prática o uso de uma tag style no cabeçalho que irá través de suas definições internas, aplicar as CSSs declaradas como padrão para todos os elementos encontrados no documento.

Estilo externo

A terceira forma de aplicação de estilos usando as CSSs é através de um documento .css dedicado a distribuir estilos globais ao "projeto". As definições desse documento -quando especificados- podem gear estilos em modo cascata.

Hierarquia de métodos

Quando ocorrer a existência de múltiplos estilos sobre um mesmo elemento, o que vai definir estilo a ser aplicado é sua ordem hierárquica no seguinte sentido:

  1. Em primeiro lugar: Estilo inline
  2. Em segundo lugar: Estilo interno
  3. Em terceiro lugar: Estilo externo
