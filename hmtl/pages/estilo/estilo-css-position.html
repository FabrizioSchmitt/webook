CSS Positions

Os presets de Position

Existem 5 tipos de position que podemos usar para "setar" o estilo dos elementos são eles:

  • Static -Posição de fluxo da página (padrão)
  • Absolute- Posição relativa ao pai desde que esse tenha posição definida.
  • Relative Relativo a posição dele mesmo (na qual ele aparece no fluxo natural dos elementos da página.).
  • Fixed
  • Stick

Static

O valor postion:static é o valor padrão dos elementos html. Não é preciso adicionar nenhum position para que todos os elementos da página contenham esse valor.por padrão, ele não pode ser movido de lugar através da alteração dos valores de atributos, seu nascimento (por assim dizer) fica na área superior esquerda acompanhando o fluxo.

Relative

O position relative é sempre relativo a ele mesmo ou seja, seu local natural no fluxo da página. Se ele for ajustado para algum ponto para além do seu original, ele vai pegar esse ponto de referência como base e se mover através dele caso sejam aplicadas modificações de localização através da página.

ABSOLUTE

Se posiciona em relação ao elemento pai, desde que o elemento pai tenha a position definida diferente de static. Caso não existe um element parent(pai) próximo ele vai procurar outro e outro e caso não houve nenhum, ele vai seguir o fluxo natural da página. É importante frisar que quando um elemento tem como position: absolute sendo child (filho) de algum elemento que, possua position diferente de static o filho vai receber como referência a posição do pai, de maneira que, qualquer modificação em seu valor de localização passará a contar a partir da posição do elemento pai.

Fixed

O elemento ajustado com sua propriedade position no valor fixed, irá ficar fixo em uma determinada posição da página, e mesmo que a mesma sofra rolagem o elemento vai ser manter no mesmo lugar, seguindo qualquer movimentação da página.
