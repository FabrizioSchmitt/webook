Estilos para texto

A estilização para texto serve tanto para modificar a aparência quanto para criar semântica onde houver marcações.
São representadas (na relação linha) por tags completas (início e fechamento)

Lista de estilizações:

Especificação de estilo. Representação sintática. Efeito Propriedade 1 Propriedade 2 Propriedade 3 Propriedade 4
Tag SMALL <small> Torna o texto pequeno
Tag EM <em></em> Texto semântico em itálico
Tag I <i></i> texto em itálico sem semântica
Tag B <b></b> Texto em negrito sem semântica
Tag STRONG <strong></Strong> Texto em negrito com semântica
Tag SUP <sup></sup> texto sobrescrito h20 22
Tag DEL <del></del> Sobre deletado
Tag MARK <mark></mark> Texto fica com marcador visual.
Tag INS <ins></ins> Texto sublinhado com semântica
Tag U <u></u> Tag sublinhada sem semântica
Tag CODE (mono-espaçado) <code></code> var=1.getNumber(23)
Tag PRE <pre></pre> Mantém a formatação original do código.
Tag Q <q></q> Citações
Tag Blockquote <blockquote></blockquote> Pode envelopar umaou mais linhas
