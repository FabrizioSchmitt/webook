Flex-Wrap

Devido ao aninhamento dos objetos em uma direção fixa, o objeto/elemento pai vai tomar para si um espaço de domínio, o qual pode ser tomando por elementos filhos em um determinado momento. Se isso acontecer, os elementos filhos poderão passar a dividir proporcionalmente o espaço delimitado. Caso não seja a intenção existe o recurso chamado: flex-wrap. Essa propriedade css permite que cada elemento filho mantenha a integridade de sua definição original em tamanho, o que pode eventualmente causar quebra de linha na página html. Nowrap é o estado original, que não mantém integridade, wrap é o valor que mantém a integridade.
