Audio

Inserir elemento áudio

O elemento áudio tem a capacidade de inserir midia de áudio para ser tocada dentro do próprio navegador. Porém, alguns navedores não irão mostrar o display dos controles de midia para audio automaticamente. É ai que entra o atributo controls o qual gera a interface de controle de audio para a página.

Multiplas fontes para 1 elemento

Por questões de incompatibilidade de navegadores pode ser que alguns não executem determinado tipo de mídia. Nesse caso é possível fornecer fontes alternativas de midia(áudio) dentro da mesma tag, envelopando as fontes dentro da tag áudio.

É importante salientar que, nessa categoria de fornecimento de mais de uma fonte de mídia, o html será executado pelo navegador na ordem em que é disposto no código fonte. Então, ele irá executar de "cima" para "baixo".
É possível ainda listar nessa ordem de execuções outros elementos que não só midias de áudio, como por exemplo, um parágrafo explicando que determinada mídia não foi executada. Caso o navegador executa alguma dessas ordens sequênciais a execução para e as instruções subjacentes não serão executadas.

Funcionalidades extras

Existem algumas funcionalidades adicionais no elemento áudio, que dizem respeito a suas funcionalidades predefinidades por exemplo.

O atributo preload define uma função de comportamento para o player de audio.

Valores para preload
Valor Funcionalidade Sintaxe
auto carrega automaticamente o arquivo de áudio e toca preload="auto"
metadata carrega informações sobre o arquivo preload="metadata"
none Não faz nada preload="none"
