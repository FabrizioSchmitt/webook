Escopos

Escopos são delimitações, metas, objetivos etc.
De modo geral, em programação escopos se referem a blocos de código, delimitados por grafismos. Em Html o escopo de uma tag começa com uma tag de abertura e uma tag de fechamento. Em linguagens de programação, esse escopo é geralmente delimitado (ou referido) como blocos e quase sempre o grafismo utilizado é { } onde o "escopo" refere-se a tudo que se encontra entre as chaves.
