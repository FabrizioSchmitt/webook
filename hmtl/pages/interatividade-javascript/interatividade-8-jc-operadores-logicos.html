Operadores binários
Sinal Tipo de proposição Condição 1 Condição 2 Comparação lógica Ordem de precedência Observação
! Negação True --- False 1 Inverte o valor lógico
&& Conjunção True False True 2 Precisa de ambos iguais para validar (esse e esse)
|| Disjunção True False False 3 Esse ou esse, precisa de um dos valores para validar

Operador ternário

O operador ternário possui 3 cláusulas. A primeira clausulá é o valor a ser avaliado. A expressão é disposta no inicio da operação (variável A) e logo após é feita uma comparação lógica.
