Variáveis em Javascript - Passo 2/Parte 1

Declarações

A forma de criar variáveis em java consiste em um dos 3 tipos de declaração:

  • var
  • let
  • const (que em verdade é uma constante não uma variável)

Seguido da declaração, é preciso definir um identificador para as variáveis declaradas. os identificadores em javascript podem não podem conter espaços, e precisam necessariamente iniciar com um dos três símbolos:

  • $
  • _
  • letra

>É preciso lembrar que javascript é uma linguagem case-sensitiva, ou seja, letras minúsculas e maiúsculas representam diferentes identificações para termos de reconhecimento; Logo, variável é diferente de VARIÁVEL ou Variável em termos de identificação.

Var

A declaração var cria espaço de locação para uma variável que pode ser de escopo global ou local, para entendermos isso melhor vamos nos aprofundar no conceito de escopo

Let

Let é uma declaração de variável apenas local

const

É a declaração de uma constante, e por consequência existe a obrigatoriedade de definir um valor inicial.
