Entidades HTML

Existem algumas "peças" em HTML que ajudam a forma o quadro que é a nossa página. Uma desssa peças são as entidades HTML, que, não são tags funcionais em lato sensu, mas, que ajudam a dar forma em nossas páginas exibindo de maneira gráfica recursos que, de outra forma, estariam disponíveis apenas processamento interno, código ou outros meios ocultos da página.
