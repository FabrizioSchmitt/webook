ANCORA E LINKS

A ancora é uma tag que permite criar ligações entre páginas de hipertexto.

Atributos

HREF

A propriedade Href, a qual é compartilhada por outras tags que não apenas a ancora a é o ponto de ligação entre locais diferentes, seja esses internos ou externos.

TARGET

A propriedade target serve para definir onde o documento alvo será aberto.

Teste com valores das propriedades target

_self

O valor _self serve que quando o link seja utilizado, ele abra na prípria página ao qual foi inserido.

_self -Link/p> Link Teste

_blank

O valor _blank serve para abrir o link em uma nova página. Link

REL

O atributo rel define qual o tipo de relação que um documento (página) tem com outro. Dos possíveis valores presentes estão:

  • author
  • external
  • prev

Download

O atributo download define qual valor/nome o arquivo alvo do link receberá como pre-definição ao efetuar a operação de download.
