Semântica geral

A divisão semântica

Estrutura de uma página A semântica de um site é utilizada para dar sentido em seus componentes/partes para além de uma mera questão visual. Ela serve para: operações de acessibilidade, aplicações de SEO, navegação interna.

Os robôs de leitura usam as tags para encontrarem conteúdo no documento, sendo assim é preciso enviar as informações precisas e para tanto precisamos utilizar as tags corretas para cada porção do site. Podemos dividir cada página html em 3 pedaços básicos: Cabeçalho, conteúdo principal e rodapé. Cada alguns desses elementos, podem conter em sí outros elementos similares. Por exemplo: um elemento conteúdo principal pode ter em si um cabeçalho e um rodapé, o outros elementos. Cada container de extra de espaço dentro de um corpo principal é o que chamamos de seção. Um elemento principal pode conter uma ou diversas seções, e cada seção pode ter seus próprios rodapés e cabeçalhos. Existe uma exceção para essa regra que á a do próprio elemento principal, pode haver apenas 1 elemento principal por documento.

Levando em consideração essas ponderações e regras, vamos iniciar a apresentação passo a passo da representação visual de uma página HTML levando em consideração a semântica como critério de marcação.

Essa tag é única por documento; Para cada subseção em um documento, para melhorar a semântica, devemos utilizar uma tag section, dessa forma a antiga div se torna uma section. <section > <header > Cabeçalho da subseção </header > conteúdo da section. E por consequência, podemos ter um rodapé para cada subseção. <footer > -Informações de rodapé. </footer > </section > </main > <footer > Rodapé do conteúdo principal. </footer >

É possível ainda que, fora do main existam assuntos que não sejam próximos o suficiente para estarem dentro do corpo principal da página distribuídos em sections, mas que ainda façam parte do conteúdo do site como um todo, nesse caso, usamos referências externas a tag main, podendo ser esses conteúdos textos complementares, imagens ou mesmo (comumente) link para outras páginas internas ou externas.
