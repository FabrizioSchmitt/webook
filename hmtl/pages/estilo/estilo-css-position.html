CSS Positions

Os presets de Position

Existem 5 tipos de position que podemos usar para "setar" o estilo dos elementos são eles:

  • Static -Posição de fluxo da página (padrão)
  • Absolute- Posição relativa ao pai desde que esse tenha posição definida.
  • Relative Relativo a posição dele mesmo (na qual ele aparece no fluxo natural dos elementos da página.).
  • Fixed
  • Stick

Static

Relative

O position relative é sempre relativo a ele mesmo ou seja, seu local natural no fluxo da página. Se ele for ajustado para algum ponto para além do seu original, ele vai pegar esse ponto de referência como base e se mover através dele caso sejam aplicadas modificações de localização através da página.

RELATIVE

ABSOLUTE

Se posiciona em relação ao elemento pai, desde que o elemento pai tenha a position definida

ABSOLUTE

Fixed

O elemento ajustado com sua propriedade position no valor fixed, irá ficar fixo em uma determinada posição da página, e mesmo que a mesma sofra rolagem o elemento vai ser manter no mesmo lugar, seguindo qualquer movimentação da página.
