Class e ID

Class

Class ou classe são atributos que podem ser inclusos na maior parte das tags e elementos html. Ele são a criados a partir de um .class que pode ser feito dentro do próprio documento html ou separado em outro arquivo como .js ou .css. Ele tem prevalência sobre elementos do tipo element. Se duplas classes forem chamados ao mesmo tempo, a prevalência na ordem de chamada será do último bloco de código inserido sequencialmente.
É possível aplicar o mesmo identificador class para diversos elementos da página e mesmo duas ou mais classes para o mesmo elemento. A ordem de prevalência para atributos classe, quando estão sobre-escrevendo algum valor será (também) pela ordem do último bloco a ser inserido sequencialmente.

ID

Ids são atributos cujo valor deve ser único. Eles tem precedência sobre qualquer outro atributo com exceção do styleInline . Ordem de prevalência. se for tag, características iguais prevalece o último adicionado na ordem emm que aparece, e sobrescreve as características predecessoras.
