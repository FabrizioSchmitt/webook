Meta-tag ViewPort

View port é uma tecnologia desenvolvida originalmente pela Apple visando criar um ajuste para navegadores mobile.
Como os browsers de mobile renderizam as páginas com uma tela virtual, muitas vezes essa tela virtual é maior que a tela do dispositivo. Então, usando o atributo content com o valor: "width=device-width justa a janela virtual para o tamanho correto do dispositivo.
