Javascript passo 4- Lidando com strings

Juntando Caracteres

Quando se pretendo misturar textos com valores ou outros pedaços de textos para formar um único bloco ou corpo de informações se faz necessário o uso de alguns recursos da linguagem em ordem de tornar esses blocos de informações dinâmicos; para isso, o javascript tem alguns recursos necessários como por exemplo a concatenação.

concatenação

Template String

Outra forma de lidar com strings é através de placeholders com valores dinâmicos os template strings. A utilização do placeholder é feita na parte interna da string utilizando a sintaxe:${variável} devidamente envolvida por demarcadores ' ' no inicio e no fim da string.
